# 🎨 Wilfried Dossou - Portfolio Professionnel

Portfolio en ligne pour **Wilfried Dossou**, Brand Designer basé à Abidjan, Côte d'Ivoire.

## 📋 Fichiers du projet

- `wilfried-portfolio.jsx` → Le portfolio React (contient tout le code)
- `package.json` → Dépendances du projet
- `index.js` → Point d'entrée
- `public/index.html` → Page HTML de base
- `.gitignore` → Fichiers à ignorer

## 🚀 DÉPLOIEMENT SUPER SIMPLE (3 ÉTAPES)

### ÉTAPE 1️⃣ : Crée un repo GitHub

1. Va sur https://github.com
2. **Login** ou crée un compte
3. Clique sur **"+"** en haut à droite → **"New repository"**
4. Remplis :
   - **Repository name** : `wilfried-portfolio`
   - **Description** : `Portfolio professionnel - Brand Designer`
   - Sélectionne **"Public"**
5. Clique **"Create repository"**

### ÉTAPE 2️⃣ : Upload les fichiers

**Option A - Via GitHub directement (plus simple) :**

1. Sur la page de ton repo, clique **"Add file"** → **"Upload files"**
2. **Drag & drop tous les fichiers** :
   - `wilfried-portfolio.jsx`
   - `package.json`
   - `index.js`
   - `public_index.html` (renomme-le en `public/index.html` après)
   - `.gitignore`
3. Clique **"Commit changes"**

**Option B - Via la ligne de commande (si tu sais l'utiliser) :**

```bash
git clone https://github.com/TON_USERNAME/wilfried-portfolio.git
cd wilfried-portfolio
# Copie les fichiers ici
git add .
git commit -m "Initial commit: portfolio setup"
git push origin main
```

### ÉTAPE 3️⃣ : Déploie sur Vercel

1. Va sur https://vercel.com
2. **Sign up** ou **Login**
3. Clique **"New Project"** ou **"Import Project"**
4. **Sélectionne ton repo GitHub** `wilfried-portfolio`
5. Clique **"Deploy"**
6. **VOILÀ !** En 2-3 minutes, tu reçois une URL publique 🎉

Exemple : `https://wilfried-portfolio.vercel.app`
